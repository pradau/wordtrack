WordTrack Help

Quick Start

  1. Select text in your Word document
  2. Click "Get Selected Text" to extract it
  3. Choose an Edit Prompt and Context
  4. Click "Send to Claude" to get AI edits
  5. Review Claude's response, then click "Insert Claude's Response"

Track Changes

To track edits made by WordTrack:

  1. Before using WordTrack, go to Review tab in Word
  2. Click "Track Changes" to enable it
  3. All WordTrack edits will appear as tracked changes
  4. Use Review tab to accept/reject changes

Spelling & Grammar

WordTrack uses Word's built-in proofing tools (not AI):

  1. Go to Review tab in Word
  2. Click "Editor" button
  3. Word will check spelling and grammar using built-in tools
  4. Review and apply suggestions as needed

Tips

  • Formatting is preserved when inserting Claude's response
  • Use "Download Debug Log" if you encounter issues
  • Context options (Personal/Formal/Documentation) guide AI tone
  • Edit Prompts are token-efficient - choose the closest match
